## Requirements

```bash
node.js >= 24.14.0
pnpm >= 10.30.1
```

## Setup

Install the dependencies:

```bash
pnpm install
```

Build Vapi AI widget into a single JS file:

```bash
pnpm build
```

Add public key and assistant ID for testing in index.html:

```bash
<vapi-widget public-key="" assistant-id=""> </vapi-widget>
```
